To use the API:

1. Create an api_key.py file inside the config directory.
2. paste your api key into that file as a string like: API_KEY = 'sk-XXXXXXX'
3. GSAS2 is a big download that I always do manually: https://advancedphotonsource.github.io/GSAS-II-tutorials/install.html
    By default it will save in your user directory. For me, they are located in "~\gsas2full\GSAS-II\GSASII"
    If your gsas2scriptable.py got installed somewhere else go into the config\constants.py file and edit that path.
